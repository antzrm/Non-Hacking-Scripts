# Non-Hacking-Scripts

A collection of utility scripts and tools not related to hacking/cybersecurity, created for personal use.

## Tools

### [ASS to SRT Converter](./ass_to_srt/)

Extracts ASS/SSA subtitle streams from MKV files and converts them to SRT format for better compatibility with media players like Jellyfin.

**Language:** Bash  
**See:** [ass_to_srt/README.md](./ass_to_srt/README.md) for detailed documentation

### [MediaInfo Format Profile Checker](./mediainfo_video_profile/)

Identifies MKV files with high-level HEVC/H.265 format profiles that may require transcoding on media servers or cause playback issues on older devices (e.g., old FireTVs).

**Language:** Python 3  
**See:** [mediainfo_video_profile/README.md](./mediainfo_video_profile/README.md) for detailed documentation
